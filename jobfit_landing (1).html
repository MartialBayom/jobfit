Pipeline

5 phases,
de la donnée au conseil.

01
Collecte & stockage
API France Travail → 6 014 offres multi-secteurs. Dédoublonnage automatique, stockage SQLite, backup CSV.
SQLAPI RESTpandas
02
Parsing CV & Embeddings
PyMuPDF extrait le texte → référentiel 50+ compétences → embeddings sémantiques 384d avec sentence-transformers (modèle multilingue FR/EN).
PyMuPDFsentence-transformersspaCy
03
Scoring ML supervisé
7 features (cosinus, Jaccard, overlap compétences…) → 3 modèles en cross-validation 5-fold → meilleur modèle sauvegardé et loggé dans MLflow.
scikit-learnMLflowGradientBoosting
04
Clustering sémantique
KMeans sur les embeddings → PCA 2D → carte interactive des offres → identification du cluster marché du profil.
KMeansPCAPlotly
05
RAG & déploiement
ChromaDB indexe les offres → recherche vectorielle → conseils personnalisés. API FastAPI + Docker + dashboard Streamlit.
ChromaDBFastAPIDockerStreamlit
